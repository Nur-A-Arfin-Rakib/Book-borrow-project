# 📚 BookBorrow — Online Book Borrowing Platform

A modern digital library platform built with Next.js, allowing users to explore, search, and borrow books online.

## 🔗 Live URL
[Coming Soon]

## ✨ Key Features
- Browse 12+ books across Story, Tech, and Science categories
- Search books by title
- Filter by category with sidebar
- User authentication (Email/Password + Google OAuth)
- Private routes for Book Details, Profile, and Update Profile
- Borrow books with toast notifications
- Fully responsive on mobile, tablet, and desktop

## 🛠️ Technologies Used
- **Next.js 15** (App Router)
- **Tailwind CSS** (Styling)
- **BetterAuth** (Authentication)
- **MongoDB Atlas** (Database)
- **React Hot Toast** (Notifications)
- **Swiper** (Carousel)
- **Lucide React** (Icons)

## 🚀 Getting Started

```bash
npm install
npm run dev
```

Create `.env.local` with:
```
MONGODB_URI=your_mongodb_uri
BETTER_AUTH_SECRET=your_secret
BETTER_AUTH_URL=http://localhost:3000
GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret
```
